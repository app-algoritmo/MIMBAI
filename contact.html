🏗 Architecture

MIMBAI uses a two-layer architecture: a static frontend on GitHub Pages and a PHP proxy server on Domeneshop that handles all Anthropic API calls securely — the API key never reaches the browser.

Browser → GitHub Pages (HTML/JS)
↓ fetch POST
api.mimbai.com (PHP proxy)
↓ curl with API key
api.anthropic.com (Claude)
↓ SSE stream
Browser renders response in real time
